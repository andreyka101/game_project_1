Репозиторий с исходным кодом 2D-аркады, разработанной на игровом движке **Godot**. Проект демонстрирует навыки проектирования игровых состояний (State Machine), работы с физикой движка, анимациями, интерфейсами пользовательского ввода и звуковым сопровождением.

В репозитории доступна полностью собранная и готовая к тестированию версия игры под платформу Android.

---

## 🛠 Технологический стек и инструменты

- **Игровой движок:** Godot Engine (работа со сценами, узлами, сигналами и физикой).
- **Язык программирования:** GDScript (объектно-ориентированный скриптовый язык, оптимизированный под Godot).
- **Сборка & Экспорт:** Конфигурация под мобильные устройства (Android SDK).

---

## 🚀 Реализованная игровая логика

- **Архитектура сцен:** Разделение игры на независимые и переиспользуемые компоненты (игрок, враги, UI, звуковые менеджеры).
- **Игровой цикл:** Обработка пользовательского ввода на мобильных экранах, отслеживание коллизий и обработка состояний победы/поражения.
- **Оптимизация ресурсов:** Эффективная работа с ассетами (аудио, спрайты) для стабильного FPS на мобильных устройствах.

---

## 📂 Структура репозитория

```text
├── ALL_scenes/            # Игровые сцены Godot (.tscn) и их скрипты логики (.gd)
├── application_apk/       # Скомпилированный готовый файл .apk для установки на телефон
├── audio/                 # Звуковые эффекты и фоновое музыкальное сопровождение
├── photo/                 # Графические ассеты, текстуры и UI элементы
├── export_presets.cfg     # Конфигурация пресетов для экспорта проекта на Android
└── project.godot          # Главный конфигурационный файл проекта Godot
```

## 📲 Как запустить игру

### Вариант 1: Быстрый запуск на смартфоне (Android)
1. Перейдите в папку `application_apk`.
2. Скачайте находящийся там установочный файл `.apk` на свое Android-устройство.
3. Разрешите установку из неизвестных источников и запустите игру.

### Вариант 2: Запуск исходного кода через движок
1. Скачайте и установите актуальную версию **Godot Engine**.
2. Клонируйте этот репозиторий:
   ```bash
   git clone https://github.com
   ```
3. В окне управления проектами Godot нажмите **Import** (Импорт) и выберите файл `project.godot`.
4. Запустите проект, нажав клавишу `F5`.
